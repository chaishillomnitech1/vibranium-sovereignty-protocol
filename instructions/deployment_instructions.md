# Vibranium Sovereignty Protocol Deployment

## Description
This repository contains the foundational files for deploying the Vibranium Sovereignty Protocol.

## Instructions
1. Clone this repository:
    ```bash
    git clone https://github.com/chaishillomnitech1/vibranium-sovereignty-protocol.git
    ```
2. Navigate to the `contracts` directory and compile the smart contract with your Solidity compiler.
3. Deploy the contract using your preferred deployment tool (e.g., Remix, Hardhat, Truffle).
4. Review the `metadata/deployment_metadata.json` file for protocol details.
